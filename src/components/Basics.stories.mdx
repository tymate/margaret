## Stack

Stack is a component that allows you to stack elements together and apply a space between them by using differents props like _direction_ and _gutterSize_.

### Example:

export const Template = args => (
  <Stack direction="column" alignX="center" gutterSize={1}>
    <Button variant="primary" {...args}>
      Hello Tymate
    </Button>
    <p>Tymate</p>
  </Stack>
);
