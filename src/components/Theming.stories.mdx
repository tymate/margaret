# Theming

The Margaret theming is based on styled-components. The theme object containes the main styles of your application. You can define the principales colors and find some utilities like spacing and font and also the default styles of the components.

## Utilities

### Spacing

The spacing allows you to customize the margin and padding scale for your application. The defined rule is that a space of 1 is equal to 16px.

##### Exemple :

```javascript
const Title = styled.h1`
  color: red;
  /* maring : 32px */
  margin: ${({ theme }) => theme.spacing(2)};
  /* padding-top : 16px */
  padding-top: ${({ theme }) => theme.spacing()};
`;
```

## Customizing

In the theme object you can add the main colors of your project. It makes easier to change the colors of the application if necessary.

##### Exemple :

```javascript
export const theme = {
  text: '#464646',
  textLight: 'rgba(0, 0, 0, 0.38)',
  background: '#f1edf2',
  primary: '#F39600',
  primaryLight: '#FFA71A',
  secondary: '#D2D213',
  error: '#F14C66',
  success: '#bada55',
};
```

It's possible to customize the button, mainNav and tabs components. You have to add the name of the componenet in the theme object and add the style you want to apply. The component can also accept a variant prop, which allows you to define stylistic variations to override the default values.

### Button

You have the exemple with the button componenet :

```javascript
button: {
    color: 'blue',
    alignX: 'center',
    fontWeight: 600,
    paddingVertical: '8px'
  },
```

### MainNav

You have the exemple with the mainNav componenet :

```javascript
mainNav: {
    width: '320px',
    maxWidth: '100%',
    boxShadow: '0 4px 4px rgba(0,0,0,0.05)',
    position: 'left',
    triggerColor: {
      expanded: '#41D974',
      collapsed: 'rgba(0, 0, 0, 0.87)',
    },
}
```
