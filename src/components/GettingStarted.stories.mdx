# Getting Started

Margaret is a library of a set of React components. It lets you quickly use custom UI components and customize them with props according to your needs.

### Installation

To get started using it, install the package and its peer dependencies with a package manager:

```
// with npm
npm install @tymate/margaret lodash react-icons react-modal react-use styled-components
```

```
// with yarn
yarn add @tymate/margaret lodash react-icons react-modal react-use styled-components
```

### MargaretProvider

To use some components of Margaret you have to add a MargaretProvider component to the root of the application. It contains some state of the app, but it also allows to apply themes to the components with passing a theme object as a prop to the provider. The theme file contains objects which hold values for common variables such as color, fonts, border, variant and more.

#### Example

```jsx
import React from 'react';
import ReactDOM from 'react-dom';
import App from './App';
import theme from './theme';
import { MargaretProvider } from '@tymate/margaret';

ReactDOM.render(
  <React.StrictMode>
    <MargaretProvider theme={theme}>
      <App />
    </MargaretProvider>
  </React.StrictMode>,
  document.getElementById('root'),
);
```

### Start

To use the components import them from the main package module:

```jsx
import { Button, Avatar } from '@tymate/margaret';
```

#### Example

```jsx
import { Button } from '@tymate/margaret';

export default const MyApp = () => (
 <Button variant="primery" fontSize="1em">Se connecter</Button>
)
```
