Typography in Margaret is based on [CSS Locks](https://blog.typekit.com/2016/08/17/flexible-typography-with-css-locks/), made available throught the theme. Once you `injectMargaret()`, you gain access to [theme.fontStyles](#theme-fontStyles) as well as the [Text](#text-component) and [shortcuts](#shortcuts) components.

## The `theme` fonts

Four elements of the `theme` object are related to typography. `fonts`,`fontStacks`, `cssLockLowerBreakpoint` and `cssLockHigherBreakpoint`. `fonts` is a collection of, _ugh_, font stacks. `fontStacks` contains all the data needed to use typography design token throughout the app (see below).`cssLockLowerBreakpoint` and `cssLockHigherBreakpoint` are respectively the names of the `viewportSizes` theme chunk lower and upper CSS lock bounds.

```js title="theme.js"
const theme = {
  ...
  fonts: {
    title: '"Archivo Narrow", sans-serif',
    body: '"Chivo", sans-serif',
    ui: 'sans-serif',
  },
  cssLockLowerBreakpoint: 'tablet',
  cssLockHigherBreakpoint: 'desktop',
  fontStacks: {
    h1Mega: {
      sizeMinRem: 1.75,
      sizeMaxRem: 2.625,
      lineHeightMin: 1.2,
      lineHeightMax: 1.1,
      fontWeight: 700,
      defaultTag: 'h1',
      fontFamily: 'title',
    },
    h1: {
      sizeMinRem: 1.375,
      sizeMaxRem: 2,
      lineHeightMin: 1.25,
      lineHeightMax: 1.2,
      fontWeight: 700,
      defaultTag: 'h1',
      fontFamily: 'title',
    },
    h2: {
      sizeMinRem: 1.25,
      sizeMaxRem: 1.5,
      lineHeightMin: 1.25,
      lineHeightMax: 1.2,
      fontWeight: 700,
      defaultTag: 'h2',
      fontFamily: 'title',
    },
    heading: {
      sizeMinRem: 1.125,
      sizeMaxRem: 1.25,
      lineHeightMin: 1.4375,
      lineHeightMax: 1.25,
      fontWeight: 700,
      defaultTag: 'h3',
      fontFamily: 'title',
    },
    subHeading: {
      sizeMinRem: 1.0625,
      sizeMaxRem: 1.1875,
      lineHeightMin: 1.2,
      lineHeightMax: 1.3,
      fontWeight: 700,
      defaultTag: 'h4',
      fontFamily: 'title',
    },
    body: {
      sizeMinRem: 1,
      sizeMaxRem: 1,
      lineHeightMin: 1.3125,
      lineHeightMax: 1.5,
      fontWeight: 400,
      fontFamily: 'body',
    },
    legend: {
      sizeMinRem: 0.8125,
      sizeMaxRem: 0.875,
      lineHeightMin: 1.4,
      lineHeightMax: 1.35,
      fontWeight: 400,
      color: 'textLight',
    },
  },
}
```

## The `Text` component {#text-component}

The Text component inherits from the Box element. The main prop it needs is the `type` prop. Its value has to be an existing key of `theme.fontStacks`. It generates the CSS lock of font size and line height, starting at `theme.cssLockLowerBreakpoint` and ending at `theme.cssLockHigherBreakpoint`, and applies default values for font-weight, color—based on theme color key, and font family—based on theme font key.

Those values can be overriden by passing `color`, `fontWeight` and `fontFamily` props, based on the same mechanics.

```jsx live
<p>En attente</p>
```

## The `theme.fontStyles` {#theme-fontStyles}

Should you need to implement font styles in more complex components or elements, you can do it through the `theme.fontStyles` object. This object is a collection of styles for each `theme.fontStack` key.

```jsx
import { Stack } from '@tymate/margaret';

const MyCustomTitle = styled(Stack).attrs({ gutterSize: 1 })`
  ${({ theme }) => theme.fontStyles.h1}

  // ☝ Those generated values can be overrided as follow 👇
  
  font-weight: 400;
  font-family: ${({ theme }) => theme.fonts.body};
  color: ${({ theme }) => theme.primary};
`;
```
