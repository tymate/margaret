The `Card` component allows to display data in a structured way.

## Props

| Name         |       description        | default |
| ------------ | :----------------------: | ------: |
| title        |         `string`         |       - |
| subtitle     |         `string`         |       - |
| fixedSize    |         `number`         |       - |
| children     |    `node` / `node[]`     |       - |
| variant      | `raised` `subtle` `link` |       - |
| alignment    |         `center`         |  `left` |
| thumbnailUrl |         `string`         |       - |
| footer       |          `node`          |       - |
