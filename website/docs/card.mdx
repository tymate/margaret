The `Card` component allows to display data in a structured way.

## Props

| Name         |       description        | default |
| ------------ | :----------------------: | ------: |
| title        |         `string`         |       - |
| subtitle     |         `string`         |       - |
| fixedSize    |         `number`         |       - |
| children     |    `node` / `node[]`     |       - |
| variant      | `raised` `subtle` `link` |       - |
| alignment    |         `center`         |  `left` |
| thumbnailUrl |         `string`         |       - |
| footer       |          `node`          |       - |

### Variant

#### _`default`_

```jsx live
<Card fixedSize={550} hasLink title="Card title" subtitle="Card subtitle">
  Card content
</Card>
```

#### _`raised`_

```jsx live
<Card
  fixedSize={550}
  hasLink
  title="Card title"
  subtitle="Card subtitle"
  variant="raised"
>
  Card content
</Card>
```

#### _`subtle`_

```jsx live
<Card
  fixedSize={550}
  hasLink
  title="Card title"
  subtitle="Card subtitle"
  variant="subtle"
>
  Card content
</Card>
```

#### _`link`_

```jsx live
<Card
  fixedSize={550}
  hasLink
  title="Card title"
  subtitle="Card subtitle"
  variant="link"
>
  Card content
</Card>
```

### ThumbnailCard

```jsx live
<Card
  fixedSize={300}
  thumbnailUrl={`https://d3v4jsc54141g1.cloudfront.net/uploads/reward_image/image/129779/Cachou_2-1454402733.JPG`}
  title="Black Cat"
  subtitle="European shorthair"
  description="The term has also been used as an elaborate way of referring to common domestic cats of Europe."
  footer={
    <Stack alignX="center">
      <Button variant="primary">More information</Button>
    </Stack>
  }
/>
```
