## Theming

The Button component accept a variant prop, which allows you to define stylistic variations to override the default values.
To add a variant to your button open the theme file (theme.js). It contains an object which holds values for common variables such as color, fonts, border, and more. In the button value you can add your variant with the style you want to apply.

Example with the variant primary:

```js title="theme.js"
const theme = {
    ...
    button: {
        primary: {
            background: 'hsl(137, 54%, 61%)',
            color: '#fff',
            boxShadow: '',

            backgroundHover: 'hsl(137, 54%, 70%)',
            colorHover: '',
            transformHover: '',
            boxShadowHover: '',

            backgroundDisabled: 'rgba(4%, 6%, 13%, 0.2)',
            colorDisabled: 'rgba(0, 0, 0, 0.38)',
            boxShadowDisabled: '',
            transformDisabled: '',

            paddingHorizontal: '24px',
            paddingVertical: '14px',
            borderRadius: '6px',
            minWidth: '220px',
        }
    }
}
```

## Props

| Name      |                              description                               |                   default |
| --------- | :--------------------------------------------------------------------: | ------------------------: |
| isLoading | <code>Displays the spinner & disables the button <br/>`boolean`</code> |                   `false` |
| variant   |    <code>Changes the appearance of the button <br/>`string`</code>     |                         - |
| disabled  |                               'boolean'                                |                   `false` |
| as        |                            `node` `string`                             |                         - |
| onClick   |                               `function`                               |                         - |
| children  |                           `node` / `node[]`                            |                         - |
| icon      |                           `node` / `node[]`                            |                         - |
| size      |                                 `full`                                 |                         - |
| fontSize  |                           `string` `number`                            |                   `'1em'` |
| loader    |                                `string`                                | `'ball-spin-fade-loader'` |
