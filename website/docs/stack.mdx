Stack is a component that allows you to stack elements together and apply a space between them by using differents props like direction and gutterSize.

```jsx live
<Stack size="full" alignX="center" direction="column" gutterSize={1}>
  <p>Hello Tymate</p>
  <Button variant="primary">Welcome</Button>
</Stack>
```
